# RSIForge Core

Minimal Bounded Self-Improving Runtime System

---

## Overview

RSIForge Core is a minimal experimental runtime designed to explore bounded adaptive behavior through:

- execution
- evaluation
- mutation proposals
- intent-bound governance
- persistent memory

The project is intentionally small, constrained, and measurable.

This repository does NOT claim to implement:
- AGI
- recursive superintelligence
- autonomous consciousness
- unrestricted self-modifying intelligence

Instead, RSIForge Core focuses on:

> the smallest possible executable loop capable of observable adaptive behavior.
